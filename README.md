## Project Summary

[![Online Store Screen Shot][product-screenshot-1]](https://devsite-humble-seed.myshopify.com/)
![Online Store Screen Shot][product-screenshot-2]

### Overview

This project was a Shopify-based online store, built in collaboration with a small team. Our goal was to customize and enhance the user experience using Shopify's Theme Kit and Liquid, while meeting the client’s tight deadline.

### Key Contributions

- [x] Customized the Debut v11.2.4 theme, including styling, block modifications, and theme functions.
- [x] Implemented Shopify's Liquid templating for dynamic content and reusable sections.
- [x] Used Shopify Theme Kit for version control and collaborative development.
- [x] Ensured a responsive, accessible, and visually appealing design.
- [x] Completed development in 3 weekly sprints, delivering on time.

### Tech Stack & Tools

* Languages: Liquid, JavaScript, SCSS/CSS
* Tools: Shopify Theme Kit, Git, VScode

### Lessons Learned

* Deepened expertise in Shopify's Liquid language and theme customization.
* Strengthened agile development skills within a small, fast-moving team.
* Gained hands-on experience with client-driven development cycles and rapid iteration.
